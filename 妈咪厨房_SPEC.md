# 孕味餐食 · 餐食推荐工具 SPEC.md

> **产品定位**：温暖治愈风的孕妈/宝妈/宝宝餐食推荐小工具
> **版本**：MVP v1.0

---

## 1. Concept & Vision

**一句话描述**：像一位贴心的营养师闺蜜，帮孕妈、宝妈、宝宝家长解决"今天吃什么"的世纪难题。

**核心感受**：温暖、治愈、专业、不焦虑。用户打开工具的那一刻，像翻开了妈妈精心准备的手写食谱本——每一餐都带着爱意和安心。

**设计关键词**：暖色调、圆润卡片、柔和阴影、温馨插画感、轻松不压抑

## 4. Features & Interactions

### 4.1 身份选择（Step 1）

**入口条件**：首次使用或点击顶部身份标签修改

**用户选项**：

| 身份 | 图标 | 标签 | 简称 |
|------|------|------|------|
| 孕妈 | 🤰 | 正在怀孕的你 | 孕妈 |
| 宝妈 | 👩 | 正在坐月子的你 | 宝妈 |
| 宝宝家长 | 👶 | 为宝宝准备餐食 | 宝宝 |

**交互**：
- 点击卡片，卡片出现选中态（橙色边框 + 缩放1.02）
- 点击"下一步"进入偏好设置
- 身份保存至 localStorage

### 4.2 偏好设置（Step 2）

**孕妈/宝妈共用**：

| 设置项 | 类型 | 选项 |
|--------|------|------|
| 忌口食物 | 多选标签 | 辛辣、海鲜、酒精、生冷、油腻、咖啡因、易过敏（自定义添加） |
| 口味偏好 | 单选 | 清淡、微辣、营养均衡 |
| 菜系偏好 | 多选标签 | 川菜、粤菜、浙菜、家常、汤品、面食 |
| 特殊需求 | 多选标签 | 补血、补钙、控糖、低脂 |

**孕妈专属**：

| 设置项 | 类型 | 选项 |
|--------|------|------|
| 孕吐情况 | 单选 | 无孕吐、轻微孕吐、严重孕吐 |

> 注：严重孕吐时，系统会过滤掉气味重、油腻、味道刺激的菜品，优先推荐清淡、易消化、止吐的食物（如苏打饼干、姜茶等）

**宝宝家长**：

| 设置项 | 类型 | 选项 |
|--------|------|------|
| 宝宝月龄 | 数字输入 | 6-36个月 |
| 每日辅食顿数 | 单选 | 1顿、2顿、3顿 |
| 过敏食物 | 多选标签 | 鸡蛋、海鲜、花生、牛奶、大豆、小麦（自定义添加） |
| 辅食类型 | 多选标签 | 泥糊状、碎末状、软固体 |

**交互**：
- 点击标签切换选中态（橙色填充/空心）
- 支持自定义添加忌口/过敏食物（输入框）
- 保存至 localStorage
- 提供"跳过"选项

### 4.3 餐食推荐（Step 3）

**推荐逻辑**（MVP阶段：静态数据+随机推荐）：

**孕妈/宝妈**：
```
早餐：主食 + 蛋白质 + 饮品
午餐：主菜 + 副菜 + 汤品 + 主食
晚餐：主菜 + 副菜 + 主食
加餐：健康小食/饮品
```

**宝宝辅食（分龄餐次）**：

| 月龄 | 每日辅食顿数 | 推荐餐次 | 说明 |
|------|-------------|---------|------|
| 6-7个月 | 1顿 | 上午添加1次 | 初次添加，以尝味道为主 |
| 8-9个月 | 1-2顿 | 上午/下午各1次 | 适应后增加频次 |
| 10-11个月 | 2顿 | 午餐+晚餐 | 可随家庭用餐时间 |
| 12-36个月 | 3顿 | 早+午+晚 | 与家人同餐，可加间食 |

> 6个月以下宝宝暂不推荐辅食食谱（纯母乳/配方奶喂养）

**孕吐过滤**（孕妈严重孕吐时）：
- 过滤：气味重的（韭菜、葱蒜）、油腻的、油炸类
- 优先推荐：清淡爽口（凉拌菜）、酸味食物（番茄）、干性食物（饼干、面包）

**单餐卡片内容**：

```
┌─────────────────────────────┐
│ 早餐                    🌅  │
├─────────────────────────────┤
│ 🍚 紫薯燕麦粥              │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│ 📝 制作方法：              │
│ 1. 紫薯去皮切块，燕麦洗净  │
│ 2. 加水煮至软糯...         │
│                             │
│ 🥗 备菜清单：              │
│ • 紫薯 100g               │
│ • 燕麦 30g                │
│ • 牛奶 100ml              │
└─────────────────────────────┘
```

**操作按钮**：
- 重新生成（随机换一套）
- 收藏（♥变红）
- 分享（复制链接/微信分享）

**日期切换**：
- 显示"今日推荐" / "明日推荐"
- Tab切换

## 6. Technical Approach

### 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **样式**：CSS Variables + Flexbox/Grid
- **数据存储**：localStorage（用户身份+偏好）
- **推荐逻辑**：静态食谱库 + 随机算法（MVP）

### 数据结构

```javascript
// 用户身份
userProfile = {
  identity: 'pregnant' | 'postpartum' | 'baby',
  preferences: {
    // 孕妈/宝妈共用
    avoidFoods: string[],
    tastePreference: 'light' | 'mild' | 'balanced',
    cuisinePreferences: string[],
    specialNeeds: string[],
    // 孕妈专属
    nauseaLevel: 'none' | 'mild' | 'severe',
    // 宝宝
    babyAgeMonths: number,
    babyMealsPerDay: 1 | 2 | 3,
    allergies: string[],
    foodTypes: string[]
  }
}

// 食谱数据
recipe = {
  id: string,
  name: string,
  mealType: 'breakfast' | 'lunch' | 'dinner' | 'snack',
  targetAudience: 'pregnant' | 'postpartum' | 'baby' | 'all',
  ingredients: string[],
  steps: string[],
  nutritionTags: string[],
  avoidTags: string[],  // 触发忌口的标签
  allergyTags: string[], // 过敏原标签
  nauseaRiskTags: string[], // 可能引起孕吐的标签
  cuisineTags: string[],   // 所属菜系
  tasteTags: string[],      // 口味标签
  babyMinAge: number,       // 适合月龄下限
  foodTypeTags: string[]   // 辅食类型（宝宝）
}
```

### 食谱库规格（v2.0）

每个角色 **50道** 食谱，按以下维度均衡分布：

**孕妈50道**：
- 餐次分布：早餐12道、午餐15道、晚餐15道、加餐8道
- 口味分布：清淡30道、微辣10道、均衡10道
- 菜系分布：家常20道、粤菜10道、川菜5道、汤品10道、面食5道
- 营养标签：补血15道、补钙10道、控糖10道、低脂10道、缓解孕吐10道

**宝妈50道**：
- 餐次分布：早餐12道、午餐15道、晚餐15道、加餐8道
- 口味分布：清淡35道、微辣10道、均衡5道
- 菜系分布：家常20道、粤菜15道、汤品15道
- 营养标签：下奶20道、补气血20道、恢复体力10道

**宝宝辅食50道**：
- 月龄分布：6-7月15道、8-9月15道、10-12月12道、12月+8道
- 餐次分布：主餐40道（早/午/晚各约13道）、加餐10道
- 辅食类型：泥糊状15道、碎末状20道、软固体15道
