# 妈咪厨房 v4.1 产品文档

> 版本：v4.1 | 更新日期：2026-05-19 | 风格：温馨清新治愈风

## 一、产品概述

**妈咪厨房**是一款面向孕妈、宝妈、宝宝家长的每日餐食推荐小程序。根据用户身份和饮食偏好，智能生成个性化一日餐食方案，支持食材匹配、菜谱采纳、日历回顾和分享功能。

### 核心价值

- **身份定制**：孕妈/宝妈/宝宝三种身份，餐食规则完全不同
- **偏好适配**：忌口、口味、菜系、特殊需求多维筛选
- **智能去重**：加权随机 + 近期历史去重，避免重复推荐
- **食材匹配**：标记已有食材，优先推荐能做的菜

```
欢迎页 → 身份选择 → 偏好设置 → 每日餐食推荐（主页面）

                                    ├── 换一批 → 重新生成
                                    ├── 菜品详情弹窗
                                    ├── 食材库 → 匹配已有食材
                                    └── 分享菜谱
```

## 三、身份与餐食规则

### 3.1 三种身份

| 身份 | ID | 角色描述 |
|------|-----|---------|
| 孕妈 | `pregnant` | 孕期营养科学摄入 |
| 宝妈 | `postpartum` | 产后滋补温和调理 |
| 宝宝家长 | `baby` | 宝宝辅食安全健康 |

### 3.2 餐食生成规则

#### 孕妈 / 宝妈（共享食谱池）

| 餐次 | 规则 | 时间段 |
|------|------|--------|
| 早餐 | 1-2道（snack类型池中轻食） | 07:00 - 09:00 |
| 午餐 | 1主食 + 1菜品 + 1汤（按recipeType分组） | 11:30 - 13:00 |
| 晚餐 | 1主食 + 1菜品（清淡不加汤） | 18:00 - 19:30 |
| 加餐 | 2道 | 15:00 / 21:00 |

- 主食判断：`isRecipeStaple()` — 粥/面/饭/馄饨/包子等
- 菜品分类：`getRecipeType()` — staple / dish / soup
- 产后恢复期自动过滤高脂肪、重口味食物

#### 宝宝

| 餐次 | 规则 | 时间段 |
|------|------|--------|
| 午餐 | 1主食 + 1菜品 + 1泥糊（9月龄+） | 11:30 - 13:00 |
| 晚餐 | 1主食 + 1菜品 | 18:00 - 19:30 |
| 加餐 | 2道（6-9月龄泥糊状） | 15:00 / 21:00 |

- 按月龄筛选：`babyMinAge` / `babyMaxAge`
- 主食判断：`isBabyStaple()` — 粥/面/饭/馄饨
- 菜品判断：`isBabyDish()` — 非主食非泥糊
- 泥糊判断：`isBabyPuree()` — foodTypeTags含'puree'

## 六、核心功能

### 6.1 智能推荐算法

```
RECIPES[identity] → 按偏好过滤 → 按mealType分组 → 加权随机选取
                                         ↑
                                    近期历史去重
                                    同主食材去重
                                    食材匹配优先
```

- **加权随机**：`weightedRandomPick()` — 近期选中过的菜谱权重降低
- **同主食材去重**：`getMainIngredientType()` — 同一餐避免全是鸡肉/全是猪肉
- **今日/明日不重复**：`generateMealsForTomorrow()` 排除今日已用ID
- **食材匹配**：`calculateIngredientMatch()` — 计算已有食材覆盖率

### 6.2 今日/明日双Tab

| 功能 | 说明 |
|------|------|
| Tab切换 | 今日 / 明日独立菜谱数据 |
| 采纳独立 | `todayAdoptedRecipes` / `tomorrowAdoptedRecipes` 分离存储 |
| 取消采纳 | 仅移除当前Tab对应日期的记录 |
| 日期映射 | `getTodayKey()` / `getTomorrowKey()` / `getTargetDateKey()` |

### 6.3 日历记录

- 采纳菜品自动写入日历：`adoptedRecords[日期key]`
- 日历弹窗支持翻月、查看某日采纳菜品
- 点击日历中的菜品可查看详情

### 6.4 食材库

- 用户录入已有食材 → `myIngredients[]`
- 菜谱卡片标注食材匹配度（绿色高亮已有食材）
- 已有食材显示为 `ingredient-owned` / `shopping-item-owned` 样式

### 6.5 分享

- 生成菜谱文本摘要，支持一键复制
- 包含菜名、食材、步骤

## 九、数据持久化

所有状态存储在 `localStorage`，key: `pregnant_meal_state`

```javascript
{
  identity: 'pregnant',                    // 当前身份
  preferences: {
    avoidFoods: [],                         // 忌口
    tastePreference: 'light',               // 口味
    cuisinePreferences: [],                 // 菜系
    specialNeeds: [],                       // 特殊需求
    nauseaLevel: 'none',                    // 孕吐程度
    babyAgeMonths: 8,                       // 宝宝月龄
    babyMealsPerDay: 1,                     // 每日辅食顿数
    allergies: [],                          // 过敏食材
    foodTypes: [],                          // 食物形态
    deliveryType: 'natural',               // 生产方式
    deliveryDate: ''                        // 生产日期
  },
  myIngredients: []                         // 已有食材
}
```

### 运行时状态（非持久化）

```javascript
{
  currentStep: 'welcome',                   // 当前步骤
  currentDay: 'today',                      // 当前Tab
  currentMeals: null,                       // 当前显示的餐食
  todayMeals: null,                         // 今日餐食
  tomorrowMeals: null,                      // 明日餐食
  todayUsedRecipeIds: null,                 // 今日已用ID
  todayAdoptedRecipes: [],                  // 今日采纳
  tomorrowAdoptedRecipes: [],               // 明日采纳
```
