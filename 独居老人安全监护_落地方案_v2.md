# 独居老人安全监护：低成本落地方案（v2.0）

> 版本说明：基于竞品分析 v2.0，核心调整——从"卖硬件"升级为"卖服务闭环"，差异化聚焦24小时值守响应

---

## 一、项目定位

### 1.1 重新定义：我们是"安全管家"，不是卖硬件的

**原定位（v1.0）**：轻量化设备+SaaS服务，报警通知子女

**新定位（v2.0）**：**独居老人安全管家——"监测+响应"服务闭环**

> 不是卖给老人一台设备，而是**为独居老人家庭提供7×24小时安全守护服务**，硬件是入口，软件是工具，值守服务才是核心价值。

**为什么这是关键升级？**

| v1.0思维 | v2.0思维 |
|---------|---------|
| 卖设备 | 卖服务（安全感） |
| 等老人出事 | 老人出事我们响应 |
| 子女收到推送自行处理 | 子女收到推送 + 我们同步响应 |
| 被小米/萤石复制 | 硬件可复制，值守服务难复制 |

### 1.2 服务闭环（全行业缺口，这是机会）

```
老人发生意外（跌倒/异常）
        ↓
设备检测/老人主动SOS
        ↓
↓ 同时发生 ↓
┌─────────────────────────────┐
│ ① 秒级推送通知子女手机APP  │
│ ② 24小时值守中心同步响应   │  ← 全行业没有这步
│ ③ 联系老人确认情况         │
│ ④ 紧急情况拨打120          │  ← 全行业没有这步
│ ⑤ 联系子女/邻居/社区       │  ← 全行业没有这步
│ ⑥ 事后跟进/记录归档        │  ← 全行业没有这步
└─────────────────────────────┘
        ↓
子女安心：不是收到一条冷冰冰的推送，而是有人在管
```

## 二、目标市场（基于竞品分析的策略调整）

### 2.1 放弃：一二线城市正面战场

原因：萤石、华为、小米全部扎堆一二线城市，流量获客成本极高，品牌打不过大厂。

### 2.2 主攻：三四线城市 + 农村

| 维度 | 一二线城市 | 三四线/农村（主攻） |
|------|-----------|------------------|
| 独居老人数量 | 多 | 极多（子女外出打工） |
| 现有监护方案 | 子女陪伴多 | 严重空白 |
| 竞争对手 | 萤石/小米/华为 | 基本没有 |
| 获客成本 | 高（线上竞争激烈） | 低（B端切入容易） |
| 社区配合度 | 低（居民分散） | 高（社区关系紧密） |
| 政策补贴 | 竞争激烈 | 竞争少，更容易拿到 |

### 2.3 目标画像

**主力用户**：子女在外打工、父母在老家独居的三四线城市家庭

**典型场景**：
- "爸在老家，我在深圳打工，每天担心他摔了没人知道"
- "妈有高血压，万一晕倒怎么办"
- "装了摄像头，但出了事我除了干着急什么也做不了"

## 三、产品方案（竞品差异化版）

### 3.1 硬件策略：不做自研，接入成熟生态

**原则**：硬件可被复制，服务闭环才是护城河。选1-2家成熟供应商代销，专注软件+服务。

| 设备 | 推荐供应商 | 采购成本 | 作用 |
|------|-----------|---------|------|
| 智能摄像头（客厅） | 萤石C6系列 / 小米 | 200-400元 | 实时查看+AI活动分析 |
| 毫米波雷达（卫生间） | 隔空科技 / RadarCare | 300-500元 | 跌倒检测，无隐私侵犯 |
| SOS一键呼叫器 | 爱牵挂 / 亲鹿 | 50-100元 | 老人主动求助 |
| 智能手环（选配） | 亲鹿 / 喵医生 | 200-400元 | 心率/定位/SOS |
| 门磁传感器 | 小米/萤石生态 | 30-50元 | 老人出门/回家提醒 |

**定价套餐（锁定竞品空白带）**：

| 套餐 | 设备 | 硬件定价 | 月订阅服务 | 年订阅 | 目标用户 |
|------|------|---------|-----------|--------|---------|
| **守护版（主推）** | 摄像头+毫米波雷达+SOS呼叫器 | **799元** | **49元/月** | **499元/年** | 普通独居老人 |
| **安心版** | 守护版 + 智能手环 | **1299元** | **69元/月** | **699元/年** | 有基础病老人 |
| **尊享版** | 全套设备 | **1999元** | **99元/月** | **999元/年** | 高净值家庭/失能老人 |

> 竞品对比：亲鹿主打中高端（产品强但无服务），萤石套装2000-5000元（贵且无响应）。我们799-1999元+订阅服务，是真正的空白带。

### 3.2 软件平台（核心体验差异）

| 模块 | 功能 | 差异化 |
|------|------|--------|
| **子女端APP** | 实时画面/跌倒报警/健康数据/值守记录 | 报警页面显示"值守中心已同步响应"，给子女真正安全感 |
| **老人端** | 一键SOS/设备配对 | 操作极简，适合老年人 |
| **值守中心系统** | 报警队列/响应记录/任务分配 | 这是核心系统，自研 |
| **社区端** | 辖区老人档案/紧急联系人/历史报警 | 社区/物业可免费使用，接入我们的体系 |

### 3.3 核心差异化功能

**① 跌倒检测 + 值守响应（闭环）**
- 老人跌倒 → 系统秒级推送子女 + 值守中心同步接单
- 值守员立即回拨老人电话确认情况
- 3次无法接通 → 自动联系子女 + 紧急联系人
- 紧急情况 → 代为拨打120并发送定位

**② "有人在管"状态页**
- 子女APP首页显示：当前值守中心在线人数、本次报警响应状态
- 对比竞品：只有"报警已推送"，我们是"值守中心王某某已接单，预计3分钟内回电"

**③ 老人活动规律学习**
- AI学习老人7天活动规律（起床/吃饭/休息时间）
- 异常时自动预警（如老人平时8点起床，今天10点还没动）
- 比跌倒检测更早发现问题

## 五、24小时值守中心（核心护城河）

> 这是区别于所有竞品的关键。也是最难被复制、护城河最高的资产。

### 5.1 运营模式选择

| 模式 | 描述 | 适合阶段 | 成本 |
|------|------|---------|------|
| **外包值守** | 托管给第三方呼叫中心 | MVP期 | 低（外包费300-800元/月） |
| **自建值守中心** | 自有团队 3-5人轮班 | 成长期 | 中（人力1.5-2万/月） |
| **AI+人工混合** | AI初筛+人工兜底 | 规模化期 | 最优 |

### 5.2 推荐路径：MVP期外包，成长期混合

**MVP期（0-100户）**：
- 托管给本地呼叫中心或连锁养老机构
- 成本：500-1000元/月
- 要求：设置专属报警队列，我们提供SOP话术

**成长期（100-500户）**：
- 自建3人值守团队（3班倒）
- 接入AI报警初筛（减少80%低优先报警的人工介入）
- 成本：约1.5-2万/月（覆盖500户）

**规模化期（500+户）**：
- AI实时响应 + 人工兜底
- AI处理常规确认，5%高危情况人工跟进
- 边际成本趋近于零

### 5.3 值守SOP（核心运营资产）

```
报警触发
  ↓
【同时进行】
① 推送子女APP（5秒内）
② 值守中心收到报警（5秒内）
  ↓
值守员回拨老人电话（1分钟内）
  ↓
┌─ 接通 ─→ 确认情况 → 正常安抚/需帮助转介
│
└─ 未接通（3次）─→ 联系子女/紧急联系人
                            ↓
                    ┌─ 紧急 → 代拨120（附定位）
                    └─ 非紧急 → 定期回访
```

## 十、关键成功因素（v2.0核心）

1. **值守响应速度**：报警后5分钟内必须有人响应，这是生命线
2. **下沉深度**：能不能进村、进社区，比技术更重要
3. **子女体验**：APP要简洁，推送要及时，响应状态要透明
4. **政府关系**：进入采购名单，就有了稳定的规模化路径
5. **本地团队**：三四线城市招本地人做服务，人脉即护城河
